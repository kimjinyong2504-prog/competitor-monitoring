# Google Colab 실행 가이드

## 빠른 시작

1. **Google Colab 열기**
   - https://colab.research.google.com 접속
   - 새 노트북 생성

2. **노트북 파일 업로드**
   - `hwasung_rna_colab.ipynb` 파일을 Colab에 업로드
   - 또는 노트북 내용을 복사하여 새 노트북에 붙여넣기

3. **셀 순서대로 실행**
   - 각 셀을 위에서 아래로 순서대로 실행 (Shift + Enter)
   - 또는 "런타임 > 모두 실행" 선택

## 실행 순서

1. **라이브러리 설치** - requests 설치
2. **API 키 설정** - 이미 설정되어 있음 (필요시 수정)
3. **DART API 모듈** - DART API 클래스 정의
4. **네이버 뉴스 API 모듈** - 네이버 뉴스 API 클래스 정의
5. **데이터 수집 실행** - 실제 데이터 수집
6. **데이터 확인** - 수집된 데이터 확인
7. **JSON 저장** - 데이터를 JSON 파일로 저장
8. **HTML 보고서 생성** - HTML 보고서 생성
9. **결과 확인** - 생성된 파일 확인 및 다운로드

## 생성되는 파일

- `hwasung_rna_data.json`: 수집된 모든 데이터 (JSON 형식)
- `hwasung_rna_report.html`: HTML 보고서

## 파일 다운로드

1. 왼쪽 파일 탐색기(폴더 아이콘) 클릭
2. 생성된 파일 우클릭
3. "다운로드" 선택

또는 코드에서 `FileLink`를 사용하여 다운로드 링크가 자동으로 생성됩니다.

## 주의사항

- Colab에서는 네트워크 제한이 없으므로 DART API가 정상 작동합니다
- API 호출 제한에 주의하세요 (DART: 일일 제한, 네이버: 25,000건/일)
- 세션이 종료되면 파일이 삭제될 수 있으므로 다운로드하세요

## 문제 해결

### DART API 오류
- API 키가 올바른지 확인
- 네트워크 연결 확인
- API 서버 상태 확인

### 네이버 뉴스 API 오류
- 클라이언트 ID와 Secret 확인
- 일일 호출 제한 확인

### 파일이 보이지 않음
- 파일 탐색기 새로고침
- 런타임이 실행 중인지 확인
