# Colab DART API 수집 문제 해결 가이드

## 문제 상황

Colab에서 실행 시:
- ❌ 기업 정보: 가져올 수 없음
- ❌ 재무 데이터: 0개 연도
- ❌ 직원 정보: 없음
- ✅ 공시 정보: 28개 (수집됨)

## 원인

고유번호 불일치:
- 사용한 고유번호: `378850` (실제로는 종목코드)
- 실제 DART 고유번호: `01532603` (공시 정보에서 확인됨)

## 해결 방법

### 방법 1: 업데이트된 코드 사용 (추천)

`hwasung_rna_colab.py` 파일이 업데이트되었습니다:
- 공시 정보에서 실제 고유번호를 자동으로 추출
- 올바른 고유번호로 기업 정보, 재무 데이터, 직원 정보 조회

**Colab에서 다시 실행:**
```python
!pip install requests -q
exec(open('hwasung_rna_colab.py').read())
```

### 방법 2: 고유번호 직접 수정

Colab 코드에서 고유번호를 직접 수정:

```python
# 기업 정보
COMPANY_NAME = "화승알앤에이"
CORP_CODE = "01532603"  # 실제 DART 고유번호로 변경
```

그리고 실행:
```python
!pip install requests -q
exec(open('hwasung_rna_colab.py').read())
```

## 주의사항

- DART API는 일일 호출 제한이 있을 수 있습니다
- 재무 데이터는 사업보고서가 제출된 연도만 수집됩니다
- 직원 정보는 최신 연도부터 조회합니다
