# Colab에서 직원 정보 수집 가이드

## 개선 사항

`hwasung_rna_colab.py` 파일이 업데이트되어 직원 정보 수집 기능이 개선되었습니다.

### 주요 변경사항

1. **직원 정보 조회 개선**
   - 최근 3년(현재 연도, 전년도, 2년 전) 동안 직원 정보를 시도합니다
   - 각 연도별로 상세한 로그를 출력합니다
   - 데이터가 없는 경우 명확한 메시지를 표시합니다

2. **에러 처리 강화**
   - 타임아웃을 60초로 증가
   - 각 연도별 조회 결과를 상세히 출력
   - 데이터 미공시 시 안내 메시지 표시

## 사용 방법

1. Google Colab에서 새 노트북을 엽니다
2. `hwasung_rna_colab.py` 파일을 업로드합니다
3. 다음 코드를 실행합니다:

```python
# 필요한 라이브러리 설치
!pip install requests -q

# 스크립트 실행
exec(open('hwasung_rna_colab.py').read())
```

또는 직접 코드를 복사해서 Colab 셀에 붙여넣고 실행하세요.

## 실행 결과

스크립트 실행 시 다음과 같은 정보가 출력됩니다:

```
[DART] 직원 정보 수집 중...
  - 직원 정보 수집 성공: 2024년
  - 직원 수: 123명 (2024년 기준)
```

또는 데이터가 없는 경우:

```
[DART] 직원 정보 수집 중...
  - 직원 정보 없음: 2024년 (데이터 미공시)
  - 직원 정보 없음: 2023년 (데이터 미공시)
  - 직원 정보: 사용 가능한 데이터가 없습니다.
```

## 주의사항

- DART API의 직원 정보는 기업이 사업보고서에 공시한 경우에만 제공됩니다
- 일부 기업은 직원 정보를 공시하지 않을 수 있습니다
- 직원 정보가 없는 경우, JSON 파일의 `employee_info` 필드는 빈 객체 `{}`로 저장됩니다

## 결과 확인

실행 후 생성된 `hwasung_rna_data.json` 파일을 다운로드하여 확인하세요.

```python
# JSON 파일 내용 확인
import json
with open('hwasung_rna_data.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
    
print("직원 정보:", data.get('employee_info', {}))
```

## 문제 해결

만약 직원 정보가 수집되지 않는다면:

1. **DART 공시 확인**: DART 홈페이지(dart.fss.or.kr)에서 해당 기업의 사업보고서를 직접 확인하세요
2. **연도 확인**: 다른 연도의 데이터가 있는지 확인하세요
3. **API 키 확인**: DART API 키가 유효한지 확인하세요
