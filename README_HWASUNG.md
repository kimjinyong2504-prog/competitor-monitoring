# 화승 R&A 실시간 분석 보고서 생성 시스템

화승 R&A의 기업 정보를 DART API와 네이버 뉴스 API를 통해 수집하여 HTML 보고서를 자동으로 생성하는 시스템입니다.

## 기능

- **DART API 연동**: 기업 정보, 재무 데이터, 직원 수, 공시 정보 자동 수집
- **네이버 뉴스 API 연동**: 관련 뉴스 기사 자동 수집 및 링크 관리
- **자동 업데이트**: 주기적으로 데이터를 업데이트하여 최신 정보 유지
- **HTML 보고서 생성**: automotive.html 템플릿을 기반으로 전문적인 보고서 생성
- **데이터 저장**: JSON 형식으로 데이터를 저장하여 이력 관리

## 설치

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 사용 방법

### 기본 사용 (한 번 실행)

```bash
python main.py
```

### 자동 업데이트 모드 (1시간마다 자동 업데이트)

```bash
python main.py --watch
```

### 커스텀 업데이트 간격 설정

```bash
# 30분(1800초)마다 업데이트
python main.py --watch --interval 1800
```

### 다른 기업 분석

```bash
python main.py --company "기업명"
```

## 출력 파일

- `hwasung_rna_report.html`: 생성된 HTML 보고서
- `hwasung_rna_data.json`: 수집된 데이터 (JSON 형식)

## 보고서 구성

생성된 HTML 보고서는 다음 섹션들을 포함합니다:

1. **기업 개요**: 기업명, 종목코드, 대표이사, 설립일, 주소
2. **재무 정보**: 매출액, 영업이익 등 최근 3년간 재무 데이터
3. **인력 현황**: 직원 수 정보
4. **공시 정보**: 최근 공시 내역 (최대 20개)
5. **관련 뉴스**: 수집된 뉴스 기사 목록 (최대 50개 표시)

## 데이터 관리

- 모든 데이터는 `hwasung_rna_data.json` 파일에 저장됩니다
- 새로운 데이터는 기존 데이터와 병합되며, 중복은 자동으로 제거됩니다
- 업데이트 이력이 자동으로 기록됩니다

## API 키

시스템에 다음 API 키가 포함되어 있습니다:

- DART API Key
- 네이버 클라이언트 ID 및 Secret

## 주의사항

- DART API와 네이버 API의 일일 호출 제한이 있을 수 있습니다
- 자동 업데이트 모드 사용 시 적절한 간격을 설정하세요
- 네이버 뉴스 API는 하루 최대 25,000건의 요청이 가능합니다

## 문제 해결

### DART API 기업 검색 실패 시

DART API는 기업명으로 직접 검색하는 기능이 제한적입니다. 다음 방법을 시도해보세요:

1. **고유번호 직접 입력**: DART 웹사이트(https://dart.fss.or.kr)에서 화승R&A의 고유번호를 찾아서 직접 입력:
   ```bash
   python main.py --corp-code "고유번호"
   ```

2. **기업명 변형 시도**: 다른 기업명으로 시도:
   ```bash
   python main.py --company "화승 R&A"
   python main.py --company "화승"
   ```

3. **수동으로 고유번호 찾기**: 
   - DART 웹사이트에서 화승R&A 검색
   - 기업 상세 페이지에서 고유번호 확인
   - `find_corp_code.py` 스크립트 실행 시도

### 네이버 뉴스 API 오류 발생 시

- SSL 인증서 오류는 회사 네트워크 환경에서 발생할 수 있습니다 (자동으로 처리됨)
- API 키가 올바른지 확인하세요
- 네트워크 연결을 확인하세요

### API 오류 발생 시

- API 키가 올바른지 확인하세요
- 네트워크 연결을 확인하세요
- API 서버 상태를 확인하세요
- 타임아웃이 발생하는 경우 네트워크 환경을 확인하세요
